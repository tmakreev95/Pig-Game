Welcome to the Pig Game 🎲

The rules of this game are simple:

1. Each player has the chance to roll the dice and hold the score by adding the current score to the score.

2. If a player rolls 1, the current score is set to 0 so think carefully before rolling the dice if you haven't hold the current score.

3. When a player's score reach 100, the active player wins the game and is able to start a new game by pressing the "New Game" button.
